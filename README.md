# GitHub User Finder

**Автор:** [Софья Кочелаевская]

## Описание
Простое GUI-приложение на Python для поиска пользователей GitHub через официальное API. Позволяет находить логины и сохранять их в локальный список избранного.

## Как использовать API
Приложение использует публичную конечную точку:
`GET https://github.com{username}`
Для работы не требуется API-ключ (при соблюдении лимитов запросов GitHub).

## Примеры использования
1. Запустите программу: `python main.py`.
2. Введите `octocat` в поле ввода.
3. Нажмите "Найти".
4. Выберите `octocat` в списке результатов и нажмите "Добавить в избранное".
5. Данные автоматически сохранятся в файл `favorites.json`.

## Тестирование
- **Тест на пустое поле:** Если оставить поле пустым и нажать "Найти", появится ошибка.
- **Тест на сохранение:** После перезагрузки приложения список избранных остается на месте.
