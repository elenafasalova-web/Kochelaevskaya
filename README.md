# GitHub User Finder

**Автор:** [Софья Кочелаевская]

### Описание
Простое графическое приложение для поиска пользователей GitHub, просмотра их информации, добавления в список избранных и сохранения этого списка в JSON-файл.

### Как использовать API
- Введите логин пользователя GitHub в поле.
- Нажмите кнопку "Поиск" для получения информации.
- Для добавления пользователя в избранные нажмите "Добавить в избранное".
- Посмотреть список избранных можно через кнопку "Показать избранных".

### Примеры использования
- Поиск существующего пользователя: введите `octocat` и нажмите "Поиск".
- Попытка поиска несуществующего пользователя — появится сообщение об ошибке.
- В избранные можно добавлять сразу нескольких пользователей и просматривать список.

### Запуск
- Убедитесь, что установлен Python 3.
- Запустите файл `main.py`.
